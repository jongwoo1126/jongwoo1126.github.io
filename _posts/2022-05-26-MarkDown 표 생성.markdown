# MarkDown 표 생성
마크다운 형식에서는 표를 생성할때 파이프 '|'를 이용한다.     
```
|마크다운|테이블|

|---|---|

|셀|셀2|
```
|마크다운|테이블|
|---|---|
|셀|셀2|

정렬은 다음과 같이 ':'를 사용한다.
```
|왼쪽정렬|오른쪽 정렬|가운데 정렬|

|:---|---:|:---:|

|셀1|셀2|셀3|

|셀4|셀5|셀6|
```
|왼쪽정렬|오른쪽 정렬|가운데 정렬|
|:---|---:|:---:|
|셀1|셀2|셀3|
|셀4|셀5|셀6|

가장 편하게 표를 생성하는 방법은 사이트를 이용하는 것이다.
